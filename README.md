## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn package manager
- A modern web browser

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Goatfighter206/Og-agent-web.git
   cd Og-agent-web
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:3000` (or the port specified in your terminal)

## ğŸ¨ Customizing Graphics

To add custom graphics, buttons, backgrounds, and other visual elements:

1. Place your image files in the `public/assets/` directory
2. Reference them in your components or CSS files
3. For detailed instructions, see [public/assets/README.md](public/assets/README.md)

Supported formats include PNG, JPEG, and SVG. PNG is recommended for images with transparent backgrounds.
